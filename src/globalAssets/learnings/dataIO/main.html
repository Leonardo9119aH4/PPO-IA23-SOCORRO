Como o nome já sugere, veremos um conteúdo sobre a entrada dos dados e também a saída de dados.

Entrada de Dados:é o comando que realiza a leitura de um dado em momento de execução do usuário. Em outras palavras, é um processo que coleta informações do usuário para que o algoritmo possa realizar suas operações.

Para fazermos o uso da entrada de dados usaremos o comando “leia( )”com a seguinte sintaxe:

Após o usuário atribuir um valor a variável a, o valor será armazenado

Saída de Dados: é o comando que realiza a exibição dos resultados gerados após a execução do algoritmo.

Para fazermos o uso da função da saída de dados usaremos "escreva( )" com a seguinte sintaxe:

escreva(<expressão e/ou variável e/ou constante>

Exemplo:

Na tela aparecerá o valor 3.

E sempre que precisar escrever algo que não venha de uma variável, deve-se fazer uso das aspas duplas(“”).

Ex:

Na tela aparecerá Olá mundo.
