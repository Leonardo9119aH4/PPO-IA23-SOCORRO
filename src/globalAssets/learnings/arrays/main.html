Matriz (Array Multidimensional)

As matrizes também podem ser chamadas de Arrays Multidimensionais. Ela tem como característica a capacidade de armazenar valores que podem ser acessados com dois índices. Para declararmos uma matriz, seguimos a seguinte sintaxe:

<Tipo> <nome_da_variável> [índice_1][índice_2]

[índice_1]: indica a quantidade de elementos que podem ser armazenados nas linhasda matriz.

[índice_2]: indica a quantidade de elementos que podem ser armazenados nas colunasda matriz.

O tamanho da matriz é dado por [índice_1] multiplicado pelo [índice_2]


Assim como explicado no Level anterior, na maioria das linguagens de programação os índices iniciam em 0 (zero), ou seja, quando é declarado um vetor de 10 elementos, o índice iniciará em 0 e terminará em 9. A única diferença que pode-se apontar é a existência de mais de índice.


Ler e Escrever a Matriz

Para escrevermos um valor de uma matriz, usaremos a seguinte sintaxe:


escreva( <nome_da_variável> [índice_1][índice_2])


Já para ler um valor de uma matriz, deve-se a seguinte sintaxe:

leia( <nome_da_variável> [índice_1][índice_2])

lembresse os índices sempre iniciam-se em 0

  • Se eu quiser exibir o número 1 na tela deve-se realizar o comando escreva( m[0][0] );
  • O comando leia(m[1][1]) exige que o usuário digite um valor de m[1][1];
  • E se quisermos que o número digitado pelo usuário seja escrito, o comando a ser usado será escreva(m[1][1] );

    • Ler e Escrever toda a Matriz

  • Escreva ao final de cada linha: ele faz a quebra da linha, sendo responsável por deixar a exibição da matriz organizada e impede com que a matriz se pareça com um vetor.
  • Leitura: leia(x[a][b]) permite atribuir um valor a cada posição do matriz “x”, de acordo com o valor atual de “a”.
  • Escrita: escreva(x[a][b]) exibe o valor de cada índice da matriz, garantindo que todos os valores armazenados sejam mostrados.
