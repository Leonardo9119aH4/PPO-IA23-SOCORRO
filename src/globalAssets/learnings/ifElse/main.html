Basicamente, o desvio condicional é uma estrutura do código que será executada somente se uma condição for verdadeira. Esse tipo de estrutura existe para tornar o algoritmo mais flexível, possibilitando diferentes trajetórias no código de acordo com as condições.

Como exemplo: “Você só consegue respirar “se” você tiver pulmões”. Isso, por si só, é uma condição.

Sobretudo, para aplicarmos os desvios condicionais usaremos o “se”, “senao” e o “senao se”. Eles são os comandos que quando uma condição for verdadeira ou falsa, serão executadas. Uma forma de vermos isso é: se() e senao se() e senao().

No código acima, o comando "se" será executado somente quando a condição for verdadeira. O comando “senao se”, será executado somente se o desvio condicional anterior for falso. E o "senao" será executado somente se a condição anterior for falsa.
