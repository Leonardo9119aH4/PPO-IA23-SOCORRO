Laços de repetição, como o nome sugere, envolve a repetição de uma determinada parte do código. Assim como os desvios condicionais, o laço de repetição depende de uma condição para determinar se o bloco de código deve ser executado, tendo sua peculiaridade que, claro, é a repetição.

Resumindo, o laço de repetição permite que um bloco de código seja repetido várias vezes.

Usos de Laços

  • Automatizar tarefas repetitivas.
  • Iterar por estruturas de dados como arrays ou listas.

Tipos de Laços

  • enquanto: executa o bloco de código enquanto a condição for verdadeira. A condição é verificada antes da execução do bloco de código.
  • faca enquanto: o bloco de comandos é executado uma vez e depois da execução é testado uma condição que caso seja verdadeira o bloco é executado novamente.
  • para: é usado quando o número de repetições é conhecido. Ela combina a inicialização de variáveis, a condição e o incremento/decremento em uma única linha.
