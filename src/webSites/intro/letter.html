O que é algoritmo?

Um algoritmo é uma série de passos feitos para executar certa ação. Em uma analogia simples, imagine o preparo de uma xícara de café.

Para preparar você precisa de água, pó de café, leite e açúcar. Todos os procedimentos que você realizar como colocar o pó de café na cafeteira ou misturar o café da xícara com leite, é o que chamamos de algoritmo, já os ingredientes é o que chamamos de entrada e a xícara de café o que chamamos de saída.

Na programação, a lógica segue essa mesma ideia de uma sequência coerente. Ela visa determinar as melhores instruções para a resolução de um problema, sendo essa sequeência um 'algoritmo'.

Formalmente falando, um algoritmo é uma estrutura de instruções bem definidas, para que ao executá-lo, tenha a solução de um problema. Para que haja essa solução, o algoritmo passará pelas seguintes fases: Coleta das informações de que o algoritmo necessita, Processamento e, por fim, a apresentação dos resultados

Antes de aprender a programar em alguma linguagem, será fundamental aprender primeiro a lógica de programação (algoritmo), para isso, será ensinado pseudocódgio (Magic-C), uma versão traduzida e simplificada do C, o Portugol.
